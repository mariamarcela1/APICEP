# Tecnologias Utilizadas
- HTML
- CSS
- React.JS
- React Router
- API VIACEP

# Como Utilizar
1. Clone o repositório para sua máquina local:
git clone https://github.com/mariamarcela1/APICEP.git

2. Entre no diretório do projeto:
cd buscador

3. Instale as dependências do projeto:
npm install

4. Inicie o projeto:
npm start
